# softwaremaker

Cursor Command：一键创建 Go + React(Vite) 前后端项目、Docker 化并推送到 GitHub。

## 使用

在 Cursor 中把本仓库的 `softwaremaker.md` 放到 `~/.cursor/commands/softwaremaker.md`，或在聊天里用 `/softwaremaker` 触发（若已配置）。

## 内容

- `softwaremaker.md`：Command 完整说明，含用途、交互步骤、可调用技能（docker-expert、golang-pro、frontend-dev-guidelines、ui-ux-pro-max、gh-cli、database、coding-agent）。
